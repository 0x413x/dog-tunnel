# Dog Tunnel(狗洞新版)
改用rust重写，集成p2p和lite版本的所有功能

支持upnp
支持本地udp固定端口
支持stun标准化服务
多端口映射
方便的动态配置
nats类型检测
支持限速
账号认证
connection抽象，可a到b，可a到c到b,中间可加filter支持脚本
静态单文件部署
方便清晰的归类配置

不保证开发进度



为什么要用rust？

- 一直想用
- 做一些技术考察，后期公司项目可能会引入
- 可更方便的进行cpu和内存精确控制
